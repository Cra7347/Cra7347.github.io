Stuff to Know:

- The maximum number of generated units per class may be no higher than 128.

- The Troll King, Baron, and Voice of El have special rituals which are tied into their respective class numbers.
The Overwrite Voice of El, The Baron, and Troll King button will overwrite their classes to make functioning random classes for them. Unfortunatly, this comes at the cost of having only one randomly generated class for each.

Install Instructions

1. Download the mod folder.

2. Set the options to your prefrences and click Save Random Classes or Overwrite Voice of El, The Baron, and Troll King, or both if you want.

3. If you are using Brave or Opera, the browser will ask you where to place the file. Place "Randomclasses.c4m" and the Randomclass folder in C:\Users\YourUserName\AppData\Roaming\coe4\mods.

That's it! Overwrite the file many times as you want.
