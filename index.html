COE 4 Class Generator

by Nurhaci

Number of Classes Generated

Choose how many custom classes you want. (Not counting the 3 generated if Overwrite Voice of El and Troll King is checked. Conquest of Elysium can only load a maximum of 75 classes, including the ones in base game.)

Maximum Number of Random Units Per Class

Maximum Number of Random Commanders Per Class - The maximum number of generated units per class may be no higher than 128.

Prevent class power upon mastery. - Changes it so random classes will levelup into units that have different ritual powers.


* The Troll King, Baron, and Voice of El have special rituals which are tied into their respective class numbers. This option will overwrite their classes to make functioning random classes for them. Unfortunatly, this comes at the cost of having only one randomly generated class for each.

Install Instructions

1. Download the mod folder here.
2. Set the options above and click Save Results or Overwrite Voice of El, or both if you want.

3. If you are using Brave or Opera, the browser will ask you where to place the file. Place "Randomclasses.c4m" and the Randomclass folder in C:\Users\YourUserName\AppData\Roaming\coe4\mods.

That's it! Overwrite the file many times as you want.
